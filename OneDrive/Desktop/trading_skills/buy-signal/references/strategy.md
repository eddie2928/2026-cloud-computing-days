## 모드 1: 일반 매수 (스캘핑)

- 기준 차트: 5분봉
- 시그널 조건 (AND):
  - 파라볼릭 SAR: af=0.02, maxAF=0.2 → 매수 신호 (SAR이 가격 아래로 전환)
  - RSI: Period=10, LPercent=30 → 과매도 (RSI < 30)
- 종목 선택: 시그널 발생 시장의 레버리지 ETF. 양쪽 동시 → 거래량 큰 쪽 우선.
- 수량: 예수금(D+2)의 50% / 현재가 = 주수 (정수 내림)
- 총 노출 한도: 예수금의 100% 초과 금지 (레버리지+인버스 합산)

## 모드 2: 물타기 매수

- 트리거: 보유 종목 평가손실률 >= -10%
- 수량: 예수금의 30% / 현재가
- 제한: 종목당 1일 1회 (averaging_down_log 확인)
- 조건 (AND):
  - 스토캐스틱 슬로우 주봉 (K=12, D=5): 매수 신호 (%K가 %D를 상향 돌파)
  - 파라볼릭 SAR 일봉 (af=0.02, maxAF=0.2): 매수 신호

## 모드 3: 인버스 헤지 매수

- 트리거: 레버리지 포지션 평가손실률 >= -5%
- 대상: 해당 시장의 인버스 ETF (코스피→252670, 코스닥→251340)
- 조건 (AND):
  - 스토캐스틱 슬로우 주봉 (K=12, D=5): 매도 신호 (%K가 %D를 하향 돌파)
  - 파라볼릭 SAR 일봉: 매도 신호 AND 스토캐스틱 슬로우 일봉 %K >= 83

## 자금 관리

- 총 거래대금: 예수금의 50% 이내
- 총 노출 한도: 예수금의 100%
- 당일 최대 손실: -2% (서킷 브레이커 발동 → 즉시 전체 매매 중단)

## 매수 차단 조건

다음 중 하나라도 true이면 매수 금지:
- trading_state.json의 `daily_target_reached` == true
- trading_state.json의 `circuit_breaker` == true
- trading_state.json의 `eod_liquidation_mode` == true
- 현재 시각 >= 15:10
- 시세 데이터 30초 이상 미수신

## LLM 보조 레이어

결정론적 조건 충족 후, LLM이 수량을 50%~100% 범위에서 조절:
- 전일 미국 시장 급락 → 수량 축소
- 외국인/기관 대규모 매도 → 수량 축소
- 악재 뉴스 → 수량 축소
- 시장 충격 이벤트 → 수량 최소(50%)

LLM 타임아웃 (5초 초과) 시: 결정론적 결과 그대로 실행 (수량 100%)
