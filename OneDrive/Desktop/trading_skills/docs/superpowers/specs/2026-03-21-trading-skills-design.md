# OpenClaw 주식 트레이딩 Agent 스킬 설계

## 개요

코스피/코스닥 레버리지·인버스 ETF 양방향 스캘핑 매매 전략을 OpenClaw Agent 스킬로 구현한다. 매수 스킬 1개 + 매도 스킬 1개, 총 2개의 스킬로 구성하며 하이브리드 방식(결정론적 규칙 + LLM 보조 판단)을 사용한다.

## 요구사항

- **플랫폼:** OpenClaw (SKILL.md 형식)
- **스킬 수:** 매수 1개 (`buy-signal`) + 매도 1개 (`sell-signal`)
- **실행 방식:** 완전 자동 (사용자 승인 없이 주문 실행)
- **범위:** 전략 로직(시그널 판단, 주문 파라미터 결정)에만 집중. 키움증권 API 연동은 별도 구현.
- **브로커:** 키움증권 Open API

### 핵심 원칙

- **결정론적 레이어가 1차 판단** — PDF의 기술적 지표 조건을 규칙으로 평가
- **LLM 보조 레이어가 2차 조정** — 참고 시그널로 매매 강도(수량 비율)를 조절하거나, 위험 상황에서 매매를 보류
- **LLM은 결정론적 레이어의 "매수/매도 실행" 판단을 뒤집지 못함** — 수량을 줄이거나 보류를 권고할 수 있음 (안전장치)

## 매수 스킬 (buy-signal) 상세

### 결정론적 레이어 — 3가지 매수 모드

#### 모드 1: 일반 매수 (스캘핑)

- **기준 차트:** 5분봉
- **조건:** 파라볼릭 SAR 매수 신호(af:0.02, maxAF:0.2) **AND** RSI 과매도(Period:10, LPercent:30)
- **대상:** KODEX 레버리지(122630) 또는 코스닥150레버리지(233740)
- **수량:** 예수금의 50% 이내에서 산출
- **거래 빈도:** 1일 평균 8~10회

#### 모드 2: 물타기 매수 (추가 매수)

- **트리거:** 보유 종목 평가손실 10% 이상
- **수량:** 예수금의 30%
- **레버리지 매수 조건:**
  - 스토캐스틱 슬로우(K:12, D:5) **주봉** 매수 신호
  - 파라볼릭 SAR(af:0.02, maxAF:0.2) **일봉** 매수 신호
  - 두 조건 동시 충족 시 실행

#### 모드 3: 인버스 헤지 매수

- **트리거:** 레버리지 포지션이 물려있을 때
- **대상:** KODEX 200선물인버스2X(252670) 또는 코스닥150선물인버스(251340)
- **조건:**
  - 스토캐스틱 슬로우(K:12, D:5) **주봉** 매도 신호
  - 파라볼릭 SAR **일봉** 매도 신호 **AND** 스토캐스틱 슬로우 **일봉** 83 이상
  - 두 조건 동시 충족 시 실행

### LLM 보조 레이어

결정론적 레이어가 매수 조건을 충족한 후, LLM이 다음을 평가하여 **매매 강도(수량 비율)를 50%~100% 범위에서 조절**:

| 참고 시그널 | 평가 내용 |
|---|---|
| 전일 미국 주식 (나스닥, S&P500) | 미국 시장 급락 시 수량 축소 |
| 외국인/기관 거래 대금 및 거래량 | 대규모 매도 시 수량 축소 |
| 데일리 시황 브리핑 | 악재 뉴스 시 수량 축소 |
| 기타 뉴스 자료 | 시장 충격 이벤트 감지 시 보류 |

**제약 조건:** LLM은 수량을 50% 미만으로 줄이거나 매수 자체를 거부할 수 없음 (최소 50% 실행 보장)

## 매도 스킬 (sell-signal) 상세

### 결정론적 레이어 — 2가지 매도 모드

#### 모드 1: 익절 매도 (기본)

- **조건:** 보유 종목의 현재가가 평균 매수가 대비 **+0.6% 이상** 상승
- **수량:** 전량 매도
- **실행:** 조건 충족 즉시 자동 매도 지시
- **대상:** 레버리지/인버스 구분 없이 보유 중인 모든 종목에 개별 적용

#### 모드 2: 목표수익률 도달 매도

- **조건:** 당일 총 실현 수익이 거래대금의 **0.4% 이상** 달성
- **동작:** 미체결 매수 주문 취소, 신규 매수 중단 신호 발생
- **보유 중인 포지션은 모드 1(0.6% 익절) 조건에 따라 개별 청산**

### LLM 보조 레이어

| 상황 | LLM 판단 |
|---|---|
| 시장 급락 감지 (지수 -2% 이상 등) | 레버리지 포지션 즉시 청산 권고 (손절) |
| 장 마감 임박 (14:50 이후) | 미익절 포지션 정리 권고 |
| 급등 모멘텀 감지 | 익절 기준을 0.6% → 1.0%로 상향 제안 |
| 대규모 외국인/기관 매도 전환 | 보유 포지션 조기 청산 권고 |

**제약 조건:**

- LLM은 0.6% 익절 매도를 지연시킬 수 없음 (도달 시 즉시 실행)
- LLM의 손절/조기 청산 권고는 별도 플래그로 매수 스킬에 전달 (매수 중단 연동)
- 익절 기준 상향 제안은 최대 1.5%까지만 허용

### 매수 스킬과의 연동

```
매도 스킬 → [당일 목표 달성 플래그] → 매수 스킬: 신규 매수 중단
매도 스킬 → [시장 위험 플래그]     → 매수 스킬: 수량 최소화(50%)
매도 스킬 → [손절 실행 알림]       → 매수 스킬: 물타기 모드 진입 판단
```

## OpenClaw 스킬 파일 구조

### 디렉토리 구조

```
trading_skills/
├── buy-signal/
│   ├── SKILL.md              # 매수 스킬 정의
│   └── references/
│       └── strategy.md       # PDF 전략 요약 (Agent 참조용)
├── sell-signal/
│   ├── SKILL.md              # 매도 스킬 정의
│   └── references/
│       └── strategy.md       # PDF 전략 요약 (Agent 참조용)
└── [원본 PDF]
```

### SKILL.md 프론트매터

```yaml
---
name: buy-signal  # 또는 sell-signal
description: "코스피/코스닥 레버리지·인버스 ETF 양방향 매매 - 매수(또는 매도) 시그널 분석 및 주문 파라미터 결정"
version: 1.0.0
metadata:
  openclaw:
    requires:
      env:
        - KIWOOM_APP_KEY
        - KIWOOM_APP_SECRET
      bins:
        - python3
    primaryEnv: KIWOOM_APP_KEY
    emoji: "📈"  # 매수: 📈, 매도: 📉
---
```

### 출력 형식 (JSON)

```json
{
  "action": "buy|sell",
  "mode": "scalping|averaging_down|hedge|take_profit|daily_target",
  "stock_code": "122630",
  "stock_name": "KODEX 레버리지",
  "quantity": 300,
  "price_type": "market",
  "confidence": 0.85,
  "reasoning": "파라볼릭 매수 신호 + RSI 과매도 확인. 미국 시장 안정적, 외국인 순매수 전환으로 수량 100% 유지",
  "flags": {
    "daily_target_reached": false,
    "market_risk": false
  }
}
```

| 당일 목표 수익률 | 거래대금의 0.4% |
| LLM 수량 조절 범위 | 50%~100% (매수), 최대 익절 1.5% (매도) |
