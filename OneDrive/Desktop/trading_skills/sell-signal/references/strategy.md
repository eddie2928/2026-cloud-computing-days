# 매도 전략 참조 문서

## 모드 1: 익절 매도

- 조건: 보유 종목 현재가 >= 평균매수가 × (1 + 익절기준)
- 기본 익절 기준: +0.6%
- LLM 사전 조정 시: 0.6% ~ 1.5% 범위
- 수량: 전량 매도
- 대상: 레버리지/인버스 구분 없이 모든 보유 종목에 개별 적용

## 모드 2: 목표수익률 도달

- 조건: 당일 총 실현 수익 >= 총 거래대금 × 0.004 (0.4%)
- 동작:
  1. trading_state.json에 `daily_target_reached: true` 기록
  2. 보유 중인 포지션은 모드 1 익절 조건으로 개별 청산 계속

## 모드 3: 장 마감 강제 청산

- 15:10: trading_state.json에 `eod_liquidation_mode: true` 기록 (매수 차단)
- 15:20: 미익절 보유 포지션 전량 시장가 매도
- 이 규칙은 모든 매도 조건보다 우선

## 모드 4: 서킷 브레이커

- 조건: 당일 실현 손실 + 미실현 평가 손실 <= 예수금 × -0.02 (-2%)
- 동작:
  1. 미체결 매수 주문 전체 취소
  2. 보유 포지션 전량 시장가 매도
  3. trading_state.json에 `circuit_breaker: true` 기록
  4. 당일 매매 종료

## LLM 보조 레이어

LLM은 0.6% 도달 전에만 익절 기준을 상향 조정 가능 (0.6%~1.5%):
- 급등 모멘텀 감지 → 1.0%~1.5%로 상향
- 0.6% 도달 후에는 무조건 즉시 매도 (LLM 개입 불가)

추가 LLM 판단:
- 지수 -2% 이상 급락 → 레버리지 손절 권고 (market_risk 플래그)
- 장 마감 임박 (14:50+) → 미익절 포지션 정리 권고
- 외국인/기관 대규모 매도 전환 → 조기 청산 권고
